<h1> コミットのルール </h1>
<h3> 先頭に[コミット内容]を記述 </h3>
・[add] ファイルの新規追加<br>
・[update] 既存ファイルの追記<br>
・[fix] 既存ファイルの修正<br>
・[remove] ファイルの削除<br>

<h1> 命名規則のルール </h1>
<h3> プライベートでも全て修飾子を付ける </h3>
・_変数名<br>
・パブリックは先頭大文字<br>
・イベントはOn●●<br>
・bool変数は_is●●<br>

<h1> その他細かいルール </h1>
・インスペクターに表示する変数には[Tooltip]、そうでないものにはsummaryを使用する<br>
